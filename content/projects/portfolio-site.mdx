## About This Site

You're looking at it! This portfolio site is built with Next.js 15 using the App Router, styled with Tailwind CSS v4, and uses MDX for content management.

## Features

- **Static Generation**: Fast page loads with static content
- **MDX Content**: Write blog posts and project descriptions in Markdown
- **Responsive Design**: Looks great on all devices
- **Minimal Aesthetic**: Focus on content, not distractions
- **Contact Form**: Server actions for form handling

## Tech Stack

- Next.js 15 with App Router
- Tailwind CSS v4
- MDX with next-mdx-remote
- TypeScript
- Deployed on Vercel

## Design Decisions

I chose a minimal design with generous whitespace and a limited color palette. The orange accent color adds warmth without being overwhelming. Typography is given priority, with Inter as the primary font for readability.
